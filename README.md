# PASSWORD GENERATOR APPLICATION

## Project Overview

The Password Generator Application is a Python-based program developed to generate strong and secure random passwords for users. The application helps users create passwords with different combinations of letters, numbers, and special characters, improving account security and reducing the risk of weak passwords.

The user can specify the desired password length, and the program automatically generates a random password using uppercase letters, lowercase letters, digits, and symbols. The application is simple, fast, and easy to use, making it useful for students, professionals, and general users who need secure passwords for online accounts and applications.

This project also helps beginners understand:

* random number generation
* string handling
* loops
* user input handling
* Python modules and libraries

The application can run directly in the Python console and provides colorful output using the `colorama` library for a better user experience.

Example:
